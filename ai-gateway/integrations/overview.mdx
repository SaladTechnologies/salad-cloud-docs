_Last Updated: April 20, 2026_

Salad AI Gateway exposes a standard OpenAI-compatible endpoint, so any tool that supports a custom base URL works with
it out of the box. Set the base URL to `https://ai.salad.cloud/v1` and use your Salad API key for authentication.
