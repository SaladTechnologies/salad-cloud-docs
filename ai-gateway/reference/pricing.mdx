## Closed Beta

During the closed beta period, Salad AI Gateway is available at a monthly flat rate, regardless of usage volume. This
covers access to all available models.

## Per-Token Pricing

The following per-token rates are provided for reference and will apply once per-token billing is enabled. Prices are
per million tokens.

| Model             | Input ($/M tokens) | Output ($/M tokens) |
| :---------------- | -----------------: | ------------------: |
| `qwen3.6-35b-a3b` |              $0.09 |               $0.60 |
| `qwen3.6-27b`     |              $0.30 |               $1.20 |
| `qwen3.5-9b`      |              $0.06 |               $0.09 |

Input tokens include your prompt, image inputs, tool definitions, and any conversation history sent with each request.
Output tokens are the generated response.
